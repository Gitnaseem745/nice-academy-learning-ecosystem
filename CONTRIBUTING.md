# Contributing to Learn in Public Ecosystem

Thank you for your interest in contributing! We want to make this the best learning resource possible.

## How to Contribute

1.  **Fork the repository**.
2.  **Create a branch** for your feature or fix.
3.  **Follow the structure**: ensure any new project follows the strictly defined `project-name` structure found in `/shared/templates`.
4.  **Use Mermaid**: if adding a new flow or explanation, use Mermaid diagrams.
5.  **Submit a Pull Request**.

## Guidelines

Please review the guidelines in `/shared/guidelines/` before submitting.
